A-side runtime is complete and has passed the current self-tests and regression tests.

Currently available interfaces:

build-class-table
lookup-class
class-exists?
make-instance
instance?
lookup-field
update-field
superclass-of
lookup-method
lookup-main-method

Two important notes:

Field access now supports shadowed fields with the same name along the inheritance chain, so the current-class parameter in lookup-field / update-field is meaningful and should not be passed arbitrarily.
update-field is purely functional: it returns a new object rather than mutating the original one in place.

Normal method lookup:

(lookup-method table runtime-class method-name '())

super method lookup:

(lookup-method table runtime-class method-name (superclass-of table current-class))

main lookup:

(lookup-main-method class-table class-name)
